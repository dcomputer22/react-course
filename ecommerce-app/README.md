# Ecommerce Application

A full-stack ecommerce application built with React (Vite) and Node.js. This project demonstrates how a frontend application interacts with a backend service to deliver a simple online shopping experience.

---

## Overview

This project was developed to strengthen my understanding of full-stack development by building a functional ecommerce system with core features such as product listing and cart management.

---

## Features

- Product listing
- Product detail view
- Add to cart
- Remove from cart
- Dynamic UI updates
- API communication between frontend and backend

---

## Architecture

### Frontend

- Built with React using Vite
- Component-based structure
- Handles user interaction and UI rendering

### Backend

- Built with Node.js
- Uses mocked JSON data instead of a database
- Provides endpoints for products and cart operations

### Communication

- REST-style API calls
- JSON used for data exchange between client and server

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/dcomputer22/react-course.git
cd ecommerce
```

### Run the backend

```bash
cd backend
npm install
node server.js
```

### Run the frontend

```bash
cd frontend
npm install
npm run dev
```

## Limitations

- Uses static JSON data (no database)
- No authentication system
- No payment integration
- Built for learning purposes, not production use
